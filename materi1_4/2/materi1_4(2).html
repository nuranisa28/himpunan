1.4 HIMPUNAN BERHINGGA DAN HIMPUNAN TAK BERHINGGA

Pelajarilah contoh soal berikut!
Diberikan himpunan sebagai berikut :
  1. A = {2, 4, 6, 8}
  2. B = {1, 2, 4, 8, ...}
  3. D = {1, 3, 5, ..., 25}
  4. C = {..., 0, 1, 2, 3}
Dari himpunan di atas, himpunan mana yang merupakan himpunan berhingga dan himpunan mana yang merupakan himpunan tak berhingga?

Penyelesaian :
  1. Himpunan Berhingga :
    • A = {2, 4, 6, 8} merupakan himpunan berhingga karena himpunan A dapat dinyatakan banyak anggotanya, yaitu n(A) = 4.
    • D = {1, 3, 5, ..., 25} merupakan himpunan berhingga karena himpunan D dapat dinyatakan banyak anggotanya, yaitu n(D) = 25.
  2. Himpunan tak Berhingga :
    • B = {1, 2, 4, 8, ...} merupakan himpunan tak berhingga karena himpunan B tidak memiliki batas sehingga yaitu n(B) = ∞.
    • C = {..., 0, 1, 2, 3} merupakan himpunan tak berhingga karena himpunan C tidak memiliki batas sehingga yaitu n(C) = ∞.
